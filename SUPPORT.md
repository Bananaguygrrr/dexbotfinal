# Support

For setup help, bug reports, data removal requests, or server support, use the official support server:

https://discord.gg/yWJHqqBRSJ

When reporting a problem, include:

- the command you used
- what you expected to happen
- what actually happened
- screenshots or message IDs if the issue is about a spawn, trade, shop listing, or inventory entry

Do not post bot tokens, API keys, passwords, or private Discord data.
