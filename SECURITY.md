# Security Policy

## Reporting Security Issues

Please report security issues privately through the support server:

https://discord.gg/yWJHqqBRSJ

Do not open public issues containing:

- Discord bot tokens
- API keys
- private user data
- exploit steps that could damage servers, inventories, coins, trades, or market listings

## Secrets

Never commit `.env` files, Discord bot tokens, Render API keys, or other credentials.

Use environment variables in Render or a local `.env` file for development.
